# 系统配置说明功能

## 功能特点

### 1. ℹ️ 信息图标
- **位置**：配置弹窗标题栏右侧
- **样式**：紫色圆形图标，带有"i"标识
- **交互**：点击切换说明显示/隐藏

### 2. 📖 配置说明内容

#### 💰 起步价
订单的基础起送费用。

#### 📏 里程费
每公里增加的配送费，系统会自动计算距离。

#### 🌤️ 天气加价
用于雨雪天气，直接附加在订单总额上，激励骑手接单。

#### 💳 抽成比例
例如设置为10%，则10元运费中平台收取1元，骑手得9元。

### 3. ✨ 交互效果

**点击"i"图标：**
- 说明区域展开（带动画）
- 再次点击收起
- 关闭弹窗后自动重置

**视觉设计：**
- 渐变紫色圆形图标
- 渐变灰色背景说明区域
- 卡片式内容展示
- 平滑的展开/收起动画

## 使用方法

### 查看配置说明
1. 打开小程序首页
2. 点击右下角配置按钮（⚙️）
3. 弹窗打开后，点击标题栏的"i"图标
4. 说明区域展开，显示详细说明
5. 再次点击"i"图标可关闭说明

### 查看配置值
- 说明区域下方是配置值列表
- 显示实际的配置数值
- 格式化显示（如：¥3元、1.2倍等）

## 代码实现

### 1. 数据结构
```javascript
data() {
  return {
    showConfigInfo: false  // 控制说明显示/隐藏
  }
}
```

### 2. 切换方法
```javascript
toggleConfigInfo() {
  this.showConfigInfo = !this.showConfigInfo
}
```

### 3. 关闭时重置
```javascript
hideConfigModal() {
  this.showModal = false
  this.showConfigInfo = false  // 重置说明状态
}
```

### 4. 条件渲染
```html
<view class="config-info" v-if="showConfigInfo">
  <!-- 说明内容 -->
</view>
```

## 技术要点

### 条件渲染
使用 `v-if="showConfigInfo"` 控制显示/隐藏

### 事件处理
- `@tap="toggleConfigInfo"` 切换显示
- `@tap.stop` 阻止冒泡

### 动画效果
- CSS `@keyframes` 定义动画
- `transition` 添加过渡效果

### 状态管理
- 弹窗状态 `showModal`
- 说明状态 `showConfigInfo`
- 关闭时同步重置
